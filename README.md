## Menu and User authority management web application
### 메뉴 및 사용자 권한처리 수행 어드민페이지
authority-management 에서 권한관련 인터셉터 적용방식을 Spring Security 적용

### 기능
- 로그인
- 메뉴 등록/수정/삭제 (권한 처리) : 조회 권한, 등록/수정/삭제 권한
- 관리자 등록/수정/삭제 (권한 처리)
- 메뉴 권한 일괄 수정

### 사용기술
- Spring Boot 3.2.x
- Spring Security 6.x
- JPA(Querydsl)
- Thymeleaf
